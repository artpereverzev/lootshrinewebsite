A ROGUELITE BUILT
AROUND LOOT CHOICES

LootShrine puts loot decisions at the center of every run. Every item you find can be equipped for instant power, absorbed for temporary buffs mid-battle, or sacrificed at the Shrine for permanent stat gains that last the run.

// THE LOOP

HOW A RUN WORKS

Every floor follows the same rhythm — but each decision changes everything.

01

BATTLE

Face waves of enemies in front and back rows. Manage the in-battle shop to gear up mid-fight. Survive.

02

SHOP

Buy, absorb for temp stats, or let it pass. Reroll the shop — costs escalate per roll.

03

SHRINE

Sacrifice items for permanent stat gains. Donate gold. Pray for divine favor. Build devotion.

04

DESCEND

Advance deeper into the tower. Enemies scale. New zones emerge. Guardian bosses every 10 floors.

05

SOUL FORGE

Run over? Spend earned souls on permanent upgrades. Grow stronger across every attempt.

// CORE MECHANIC

EVERY ITEM IS A
THREE-WAY DECISION

This single mechanic is the heart of LootShrine. No two runs feel the same.

01 / 03

EQUIP

Slot the item into your build for immediate stat bonuses. Complete sets for powerful bonuses — but only 3 active sets at once forces real choices.

02 / 03

ABSORB

Consume the item for a temporary stat surge this battle. No turn cost — burn it for burst power when you need an edge right now.

03 / 03

SACRIFICE

Feed the shrine altar. Gain a fraction of the item's stats permanently for this run. Higher-rarity sacrifices yield greater gains. Critical sacrifices double the reward.

// GAME FEATURES

BUILT FOR DEPTH

Every system designed to reward mastery and encourage experimentation.

TURN-BASED COMBAT

No time pressure. Think through every action. Attack, defend, use skills, or manage your in-battle shop.

ROW-BASED ENEMIES

Up to 6 enemies in front/back rows. Front row falls, back row promotes. Target priority matters.

SET BONUSES

Collect matching items to activate set bonuses. Only 3 sets can be active — choose your synergies carefully.

SKILL TREES

Unlock nodes each run, reset on death. Class-specific trees define your combat style each attempt.

DIVINE ITEMS

Ultra-rare boss-only divine loot with unique mechanics that break the rules. Cannot be sacrificed — only equipped.

MERCHANT SYSTEM

The Shadow Peddler follows you through the tower, leveling up across runs with quests and rare artifacts.

ITEM MERGING

Merge same-tier items to climb the rarity ladder: Legendary, Mythic, Cosmic. The highest tier items defy imagination.

75+ ACHIEVEMENTS

Full Game Center integration with achievements across progression, combat, collection, and hidden milestones.

GHOST CHALLENGES

Challenge friends with shareable run codes. Race ghost runs and compare your builds head-to-head.

64 LEADERBOARDS

Global and per-class Game Center leaderboards. Race for the highest floor, most enemies slain, or fastest runs.
